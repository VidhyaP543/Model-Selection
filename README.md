# Model Selection and Evaluation

## Project Overview
This project compares different machine learning models and selects the best one using evaluation metrics and cross-validation.

---

## Objective
- Train multiple ML models
- Compare their performance
- Use evaluation metrics
- Apply cross-validation
- Select best model
